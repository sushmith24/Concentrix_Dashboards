# Concentrix_Dashboards
Data analytics portfolio with dashboards and VOC insights.

**1. Amazon sales dashboard**
Analyzed customer and sales data to identify trends, categorize patterns, and generate insights for business decision-making.

**2. Urban Bliss dashboard**
Analyzed urban happiness data to identify key factors such as traffic, air quality, cost of living, and healthcare impacting city well-being. Built a dashboard to explore trends and compare city-level performance for better decision-making.
